Quick Start Guide

1. Upload a SCORM Package:

POST /api/packages/upload with form-data containing the .zip file


2. Launch a Course:

GET /api/packages/:packageId/launch to get the launch URL


3. Track Progress:

Use SCORM API endpoints at /api/scorm/* for session management


4. Download for Offline:

GET /api/packages/:packageId/download to get an offline bundle
